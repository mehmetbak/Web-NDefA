User Guide | Web-NDefA: Web-'N'etwork 'Def'ormation 'A'nalysis

Please read the terms of use carefully before using our site.

  • This site is a web-based open-source platform (Web-NDefA) developed for the determination of deformations in geodetic networks, which are periodically monitored with GNSS observations, with static model and S-Transformation matrix method for easy access to GNSS-based deformation analysis studies.
  • The information and documents submitted as a result of the inquiry are for informational purposes only. It cannot be used for official transactions and commercial purposes. In case of misuse, all responsibility belongs to the people who use it inappropriately.
  • Users cannot in any way claim that they have suffered damage due to incorrect information on the site.
  • The admin reserves the right to change or terminate the content of the site or any service provided to users at any time.
  • The admin is working to make the site accessible. However, it is not responsible for the inaccessibility of the site for various reasons.
  • For the Web-NDefA Application to work, the baseline solution files of a GNSS-based geodetic network must be uploaded to the platform in .asc or .txt format.
  • The file to be uploaded to the platform is user-defined and the keywords and information in the file are given below.
    • @+ Individual baseline information (Reference point of baseline and its coordinates).
    • @- Baseline vector components.
    • @= Variance-covariance information for baseline vector.
    • REF: Reference point and coordinates.
    • MEAN: Rover point and coordinates.
  • The template baseline solution file can be downloaded from the platform.
